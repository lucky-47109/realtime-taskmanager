# Real-Time Task Manager (React)

A lightweight task manager built with React.  
Supports boards, columns, and tasks with instant UI updates.

---

## Features
- Board → Column → Task structure
- Fast state-driven updates
- No page refresh required
- Clean, scalable React components

---

## Tech Stack
- React 18
- JavaScript (ES6+)
- CSS

---

## Run Locally
```bash
npm install
npm start
````

Open: `http://localhost:3000`

---

## Notes

* Updates are live within the same tab.
* Cross-tab / multi-user sync requires WebSockets or a backend.

---

## Future Scope

* Drag & drop
* Persistent storage
* Real-time multi-user sync
